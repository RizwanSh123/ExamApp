📖 How Chapter Weightage Works
1. Select a paper (dept + subject) then add chapters with marks.
2. Tag each question to a chapter when adding questions.
3. Students get questions picked proportionally from each chapter's pool.
4. Example: 3 chapters (10M+5M+15M=30M paper) → Ch1 contributes 33%, Ch2 17%, Ch3 50%.
